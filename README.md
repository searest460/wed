# Wedding Invitation: Andrea & Pedro

A beautiful, interactive wedding invitation web application.

## 🚀 Presentation Options

### 1. View Live (Vercel)
The project is configured for deployment on Vercel. Simply push to your GitHub repository and it will deploy automatically.

### 2. Static Preview (Offline)
The project is fully portable. To view it without a server:
1. Open the `web` folder.
2. Double-click `index.html`.
3. The invitation will load with all local assets (video, music, fonts).

## 🛠 Development Setup

If you wish to edit the source code:

1. **Install Node.js**: Ensure you have Node.js installed.
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run Development Server**:
   ```bash
   npm run dev
   ```
4. **Build for Production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `index.html`: Main entry point (currently set to use pre-compiled assets for client preview).
- `src/`: React source code and components.
- `public/`: Static assets including fonts, videos, and music.
- `assets/`: Pre-compiled JavaScript and CSS files.

## 🎨 Customization
You can edit the placeholder components in `src/components/wedding/` to build out the full React version of the site.
